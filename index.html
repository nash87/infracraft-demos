Which Edition

Use parkhub-docker when

  • ✓You have a VPS, home server, or Kubernetes cluster
  • ✓You want one Docker command to get running
  • ✓You need maximum performance — thousands of users
  • ✓You want zero runtime dependencies and no DB server
  • ✓You deploy on Kubernetes, Nomad, or bare metal

Use parkhub-php when

  • ✓You're on shared hosting — cPanel, Plesk, IONOS
  • ✓You want free-tier PaaS hosting (Render, Railway)
  • ✓You can't install Docker on your server
  • ✓You already run PHP + MySQL infrastructure
  • ✓You need browser-only setup — no CLI required
Architecture

Rust Stack

Axum web framework, redb embedded database, Argon2id auth. Single static binary — compiles to ~30 MB with the SPA bundled in. No runtime except musl libc.

PHP Stack

Laravel 12 with Sanctum auth, MySQL or SQLite. Composer + Node build. Deploys on any PHP 8.2+ host, from shared cPanel to Docker to free PaaS tiers.
